🎬 AI Personalized Video Generator

A fully automated pipeline that identifies high-priority leads, writes a personalized video script using Claude, generates an AI video via Visla, and delivers the video link — running on autopilot every 5 minutes.

The Problem

Video outreach has among the highest reply rates in sales, but recording a personalized video for each prospect takes hours. Most teams either skip it or send the same video to everyone, defeating the point.

The Solution

An n8n workflow runs every 5 minutes, pulls high-pain-score leads from Supabase, sends their data to Claude to generate a tight 45-second video script, then calls the Visla API to create the video. Once Visla finishes rendering, the workflow polls for completion, grabs the video link, and saves it back to the database.

Step by Step
⏱️

Schedule Trigger — Every 5 Minutes

The workflow checks for new high-pain-score leads in Supabase that haven't had a video generated yet.

🗄️

Pull Leads from Supabase

Fetches leads marked as high priority. Each lead includes company context, pain signals, and contact info.

🧠

Claude Writes the Script

A detailed prompt instructs Claude to write a 45-second personalized video script. It references the lead's specific pain point and keeps the tone conversational.

🔐

HMAC Auth for Visla API

Visla requires HMAC-signed requests. The workflow generates the signature dynamically before each API call.

🎬

Visla Creates the Video

The script gets sent to Visla's API which generates an AI presenter video with the personalized script.

🔄

Poll Until Complete

The workflow polls Visla every 30 seconds until the video is rendered — handles variable processing times gracefully.

💾

Save Video Link to Supabase

Once the video URL is ready, it's saved back to the lead record in Supabase for the sales team to use.
